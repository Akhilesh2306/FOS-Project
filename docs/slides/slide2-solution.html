The Solution
Four features that ship together.
A proactive AI sales assistant. Not a chatbot. A system that monitors deals, detects problems, generates recommendations, drafts content, and learns from feedback.
1
Next Best Action
The system tells the rep exactly what to do next for a given deal. Detects triggers (stale deals, stage changes, MEDDPICC gaps), generates 3 to 5 ranked recommendations with justifications.
Delivered via Slack DM + Dashboard
2
Dynamic Playbook
A custom 11-section sales guide for every deal. Adapts to industry, stage, competitors, and stakeholders. Enriched with battle cards and case studies from the content library.
Delivered via Dashboard + Slack on demand
3
Slack Chat
Reps ask questions in Slack: deal health, meeting prep, competitor intel. The system responds with structured analysis. Also sends proactive notifications when deals need attention.
Primary interface for Account Executives
4
Web Dashboard
Visual interface for AEs (recommendations, health scores, content approval) and Managers (pipeline overview, team metrics, coaching opportunities, MEDDPICC tracking).
Web application
Streamlit as test harness: A separate Streamlit app is used during development to validate NBA generation, playbook output, and health scoring without waiting for the web dashboard or Slack bot integration.
