Architecture rule: AI agents query Semantic views only. They never access Gold, Silver, or Bronze directly. This creates a clean contract between the data team (who owns Gold) and the AI team (who builds agents). If Gold logic changes, Semantic absorbs the change without requiring agent code updates.
