AI Agents
Five specialists. One coordination pattern.
Each agent has a single responsibility. They coordinate through an event queue and Azure Event Hubs.
Detection
The Monitor
Trigger
Snowflake Task every 60 min + Event Hubs for real-time
What It Does
Scans all open deals for: stale activity, stage changes, MEDDPICC gaps, health drops, close date movements
LLM
None. Purely data-driven.
fos_agent_events
NBA
The Advisor
Trigger
Events from Detection Agent via Event Hubs
What It Does
Pulls deal context from v_opportunity_llm_context, builds prompt, calls Cortex LLM, generates 3 to 5 ranked recommendations
LLM
Snowflake Cortex LLM
fos_next_best_actions
Playbook
The Strategist
Trigger
Stage change or on-demand request
What It Does
Generates 11-section deal guide tailored to industry, stage, competitors, personas
LLM
Snowflake Cortex LLM
fos_playbooks
RAG
The Researcher
Trigger
Called by NBA and Playbook agents
What It Does
Two-stage retrieval: metadata filter (SQL) then vector similarity (Cortex Search)
Content
Battle cards, win/loss reports, case studies, competitive intel
fos_content_chunks
Feedback
The Observer
Trigger
User interaction (thumbs up/down, ratings)
What It Does
Collects feedback, tracks whether actions led to stage progression or wins
LLM
None. Data collection only.
fos_feedback_log
How They Coordinate
1
Detection scans
Snowflake Task runs every 60 min, scans via Semantic views
→
2
Events published
Triggers written to Hybrid Table + published to Azure Event Hubs
→
3
Agents consume
NBA, Playbook, RAG subscribe to Event Hubs topics
→
4
Process and write
Agent queries Semantic, calls Cortex LLM, writes results to Hybrid Tables
→
5
Deliver
Results sent via Slack DM or shown on dashboard
Snowpark Container Services (hosting) LangGraph 1.0 (orchestration) langgraph-checkpoint-snowflake (state) Snowflake Cortex LLM (inference) Cortex Search (RAG) Azure Event Hubs (pub/sub) Azure Container Apps (as needed) LangSmith (tracing)
Feature to Agent Mapping
Next Best Action
Detection Agent (finds triggers) + NBA Agent (generates recommendations)
Dynamic Playbook
Playbook Agent (generates guide) + RAG Agent (finds relevant content)
Slack Chat
Routes to the right agent: health queries to NBA, content to RAG, playbooks to Playbook
Web Dashboard
Reads from all agent output tables. Feedback Agent serves all features.
